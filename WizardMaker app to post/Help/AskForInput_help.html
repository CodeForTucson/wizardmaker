Help for the Ask for Input Element Page

You only need to create this element if you want to perform a calculation for the user such as; ingredients for a recipe depending on number of servings, the amount of soil for a container garden depending on the dimensions of the container, or how much lumber to buy depending on the height of a bookcase.

The calculations are done in the Text element but the Ask for Input element must appear first. If the Text element does not find the information, the text will not appear.

The result of creating an Ask for Input element is an input form in the wizard so you must provide two pieces of information.

  1. A label for the input field
  2. A name for the information you receive (use only letters and numbers -- no special characters or spaces). Just make up a short name that you will remember
  3. Then select Save to save the information
