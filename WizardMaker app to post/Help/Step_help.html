Help for the Step Page

Enter or change the title of your step. Select + to add elements to your step such as text, pictures, video and input fields (asking for input). If elements are already there because you are returning to this page, just select an element (one of the long light blue buttons) to change it.

Remember, if you want to display calculations you need to add one or more Ask for Input elements to the steps and collect the number from users before using them in a text element. If the text element has no calculations then you don't need any Ask for Input elements.

You can reorder the elements in your step or delete them by first selecting the large blue Move/Delete button. Red icon buttons will appear to the left of each element. Select one to move or delete an element.

When you select one of these red icon buttons you will see a pop-up window with three options. You can move the element up or down or delete it.


Select the Preview button to see how this step will look to the end user.
