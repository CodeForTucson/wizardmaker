Help for the Elements Page

Select the type of element you want to add to the page.

Text. You can write lines, paragraphs or lists of items. You can also display numbers that change depending on what number the user has provided. To get this number from the user use Ask for Input below.

Picture or Video. You can upload any jpg or png file to your page or a video in mp4 format. If your movie is in a different format, see the Help page for Picture or Video for suggestions on how to convert your file.

Ask for Input. This allows you to add a label and input field so that users can add a number to be used in the calculations in the Text element. If you need more than one number just add multiple Ask for Input controls. Be sure these elements precede the Text element that will use them. You can ask for the numbers on one step and use them on that same step or any later step.
